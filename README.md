# Guessing-number
building a game where the user keeps guessing a number until they get it right.
Beginner Guessing Game – Instructions (No Code)
Objective

Build a game where the user keeps guessing a number until they get it right.

Step 1: Define the Goal
The program should have a secret number
The user will try to guess it
The program gives feedback after each guess
Step 2: Set Up the Secret
Choose a fixed number (for now, don’t make it random)
This is the number the user is trying to guess
Step 3: Get User Input
Ask the user to enter a number
Make sure the input is treated as a number, not text
Step 4: Create the Loop
The program should keep running until the correct guess
Think: “repeat until correct”
Step 5: Add Logic Inside the Loop

After each guess:

If the guess is lower than the secret number → tell the user it's too low
If the guess is higher than the secret number → tell the user it's too high
If the guess is correct → tell the user they won
Step 6: Control the Loop Exit
The loop should stop only when the guess is correct
Think about what condition makes the loop end
Step 7: Improve the Experience (Optional)
Count how many attempts the user makes
Show the number of attempts when they win
Handle invalid input (like letters instead of numbers)
 Key Thinking Questions (don’t answer for him)
What condition keeps the loop running?
What changes every iteration?
How does the program know when to stop?

This is perfect beginner territory—loops, conditions, and state.
